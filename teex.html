Teex - Terminal Text Editor

About Teex

Teex is a terminal-based text editor written in C using the ncurses library. It is a work in progress, developed in my free time, and aims to provide a lightweight and efficient text-editing experience directly in the terminal.

Features

  • Help window (Ctrl+H)
  • Basic text editing: inserting, deleting, and moving text
  • File saving (Ctrl+S)
  • Undo recent actions (Ctrl+Z)
  • Redo restored characters (Ctrl+Y)
  • Arrow key navigation
  • Open and edit existing files via the command line
  • Exit with the Esc key
